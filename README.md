# Fastera Pages

Public GitHub Pages site for Fastera.

Planned repository:

- public repo: `ahmederaky2-cmyk/fastera-pages`
- expected Pages URL: `https://ahmederaky2-cmyk.github.io/fastera-pages/`

Includes:

- landing page
- privacy page
- support page
- lightweight static assets for App Store-facing links

Deployment:

1. Create a public GitHub repository named `fastera-pages`.
2. Push this folder as the repository root.
3. Enable GitHub Pages from the `main` branch root.
4. Use the resulting URL for:
   - Marketing URL
   - Support URL
   - Privacy Policy URL
